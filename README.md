# Tumminello Search Demo

Versione web statica della demo interattiva Tumminello, pronta per GitHub Pages.

## Come provarla in locale

Apri `index.html` nel browser, oppure avvia un server statico dalla cartella:

```bash
python3 -m http.server 8080
```

Poi apri:

```text
http://127.0.0.1:8080
```

## Come pubblicarla su GitHub Pages

1. Crea un nuovo repository GitHub, per esempio `tumminello-demo`.
2. Carica dentro il repository tutti i file di questa cartella.
3. Vai in `Settings > Pages`.
4. Scegli `Deploy from a branch`.
5. Seleziona branch `main` e cartella `/root`.
6. Apri l'URL generato da GitHub Pages.

## Comandi durante la presentazione

- Click su personas, step e CTA laterali.
- Freccia destra, freccia giù, spazio o PageDown: avanti.
- Freccia sinistra, freccia su o PageUp: indietro.
- Tasti `1`, `2`, `3`, `4`: cambio persona.
- Tasto `F` o bottone `Fullscreen`: schermo intero.
